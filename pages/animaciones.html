Base

En adelante, para demostrar las animaciones utilizaré los siguientes parámetros del elemento como configuración base, siempre que no haya una sobrescritura dentro del código. 

               
.--element{
   animation-duration: 5s;
   animation-iteration-count: infinite;
}

}

   }
   to {

animation-direction

   background-color: var(--banana-cream);
   animation-name: slide;

   animation-direction: normal;
}

.--slide .--element__reverse {
   background-color: var(--deep-blueberry);
   animation-name: slide;

   animation-direction: reverse;
}

.--slide .--element__alternate {
   background-color: var(--warm-caramel);
   animation-name: slide;

   animation-direction: alternate;
}

.--slide .--element__alternate-reverse {
   background-color: var(--ice-mint);
   animation-name: slide;

   animation-direction: alternate-reverse;
}
